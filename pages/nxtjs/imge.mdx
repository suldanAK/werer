# Uploadthing inage uploads 

## installation 

 ```hash copy filename=">Terminal "
npm install uploadthing

```

`Uploadthing` waa adeeg sahlan oo u oggolaanaya isticmaaleyaasha inay si fudud u abuuraan adeegyo ama API-yada lagu kaydiyo faylasha. Waa qalab loogu talagalay horumariyeyaasha 
(developers) si ay si fudud ugu daraan shaqada dirista iyo kaydinta faylasha codsiyadooda. Waxay la jaanqaadi kartaa teknoolojiyado kala duwan sida Next.js, si loo abuuro nidaam 
fudud oo faylalka loo kala wareejin karo loona kaydin karo.

## Faa'iidooyinka Uploadthing:

### Fududeynta 
shaqada kaydinta faylasha: Waxay ka dhigeysaa hawsha isku dhufashada iyo dirista faylasha mid aad u fudud, gaar ahaan marka la isticmaalayo frameworks-ka casriga ah 
sida Next.js iyo kuwa la midka ah.
### Waqti badbaadin: 
Horumarinta API-gaaga u gaar ah ee faylalka waa hawl culus oo waqti badan qaadata. Uploadthing waxay kuu fududeyneysaa inaad diiradda saarto dhinacyada kale ee horumarinta
 codsigaaga.
### Ammaan iyo isku haleyn:
 Waxay bixisaa ilaalin amni leh oo faylasha la xakameeyo, iyadoo ka hortagaysa in faylasha sirta ah si fudud loo helo ama loo lumiyo.
La jaanqaadista codsiyada kala duwan: Waxay ku shaqeyn kartaa barnaamijyo badan oo kala duwan, sida web apps ama mobile apps, iyadoo si fudud loogu daro API-yadooda.
Isku-darka xawaare sare leh: Waxaad si fudud ugu dari kartaa Uploadthing koodkaaga, taasoo ka dhigan inay si hufan ugu shaqeyn karto mashruuc kasta oo aad ka shaqeyneyso.
Sidaas darteed, Uploadthing waa xal lagu horumarinayo shaqooyinka kaydinta faylasha iyada oo la fududeeyo loona soo hormariyo waayo-aragnimada isticmaalaha iyo
horumariyaha labadaba.

##  generateSlug

Kani waa function loo yaqaan generateSlug, kaas oo loogu talagalay in uu abuuro slug oo nadiif ah, oo ku saleysan ciwaan la bixiyay. Slug-gu waa qayb ka mid ah URL-ka kaas 
oo badanaa loo adeegsado in lagu muujiyo magac ama ciwaan si uu u noqdo mid ku habboon akhristayaasha iyo raadinta (SEO).
Faahfaahinta Qaybaha Koodka:
1.	Function-ka generateSlug:



 ```hash copy filename=">Terminal "
export function generateSlug(title) { ... }
```

o	title: Waa ciwaanka ama qoraalka asalka ah ee la doonayo in laga abuuro slug.
o	slug: Waa natiijada ugu dambeysa ee loo sameeyo iyada oo la marsiinayo tillaabooyin kala duwan.

2.	Tillaabooyinka lagu abuurayo slug:


 ```hash copy filename=">Terminal "
const slug = title
  .toLowerCase() // Convert the title to lowercase
  .replace(/\s+/g, "-") // Replace spaces with dashes
  .replace(/[^\w\-]+/g, "") // Remove non-word characters except dashes
  .replace(/\-\-+/g, "-") // Replace multiple consecutive dashes with a single dash
  .replace(/^\-+/, "") // Remove dashes from the beginning
  .replace(/\-+$/, ""); // Remove dashes from the end
o	.toLowerCase(): Waxa ay ciwaanka oo dhan ka dhigtaa xarfo yaryar, si ay slug-gu ugu ekaadaan mid isku mid ah, xataa haddii xarfo waaweyn lagu qoray.
o	.replace(/\s+/g, "-"): Waxay boosaska bannaan ee ciwaanka ku beddeshaa dashes -, taas oo ka dhigaysa URL-ka mid waxtar leh oo ku habboon isticmaalka.
o	.replace(/[^\w-]+/g, ""): Waxay meesha ka saartaa dhammaan calaamadaha aan ahayn xarfo, tirooyin, iyo dashes, taas oo hagaajinaysa nadaafadda slug-ga.
o	.replace(/--+/g, "-"): Haddii dashes badan ay isku xigaan, waxay ku beddeshaa mid kaliya, si looga fogaado wax aan fiicnayn.
o	.replace(/^-+/, "") iyo .replace(/-+$/, ""): Waxay meesha ka saaraan dashes-ka ku yaal bilowga iyo dhammaadka slug-ga, si natiijada ugu dambaysa ay u ahaato mid 
nadiif ah.
```

3.	Natiijada Function-ka:


return slug;
o	return slug: Waxaa dib loogu soo celinayaa slug-ga ugu dambeysa ee la hagaajiyey, kaas oo loo isticmaali karo URL-ka ama si kale.
Shaqada: Function-ka generateSlug wuxuu u oggolaanayaa in ciwaan kasta laga abuuro slug nadiif ah oo ku habboon isticmaalka URL, taas oo wanaajinaysa
 SEO iyo habsami u socodka isticmaalka. Tusaale ahaan, haddii title uu yahay` "Hello World! This is a Test",` slug-gu wuxuu noqon doonaa `"hello-world-this-is-a-test".`
